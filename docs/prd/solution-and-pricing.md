# Resonance Solution and Initial Pricing

## 1) Solution Overview
Resonance is a model-agnostic AI workbench for knowledge workers, with a strong fit for consultants and small project teams.

It combines:
- Structured markdown note-taking.
- An AI chat workspace with long-term memory and project context.
- Direct terminal access for command execution and system/API integration.
- Python scripting support for analysis, reporting, and automation.

The core value is one focused environment where users can think, execute, document, and iterate without switching tools.

## 2) Core Product Capabilities
- Integrated Markdown Editor:
  - Central workspace for notes, decisions, and deliverables.
- AI Chat with Long-Term Memory:
  - Maintains continuity across sessions and work packages.
  - Reuses prior context to reduce repetition.
- Terminal and API Execution:
  - Connects to existing workflows through CLI and APIs.
  - Enables operational tasks directly from the workbench.
- Custom Python Scripting:
  - Supports data preparation, analysis pipelines, and automation.
- Model Agnostic Architecture:
  - Users can choose preferred LLM providers.
  - Reduces lock-in and supports privacy/compliance choices.
- Local and Enterprise Hosting Options:
  - Individual local usage.
  - Enterprise deployment in customer-controlled environments.

## 3) Product Positioning
Resonance is positioned as a focused "workbench mode" application:
- Single installable app.
- Reduced tool fragmentation.
- Better continuity between planning, execution, and documentation.

Strategic differentiator:
- Enterprise-grade flexibility without forcing one model provider.

## 4) Access and Protection Model
Product policy:
- Standard VS Code functionality remains available.
- Resonance extension capabilities require login before protected actions can be used.

Why this matters:
- Clear account boundary for paid usage and support.
- Reliable enforcement for hosted services and enterprise controls.
- Consistent user experience across chat, tools, and account features.

## 5) Initial Pricing Ideas

### Tier A: Individual / Small Business
- Standalone Workbench License:
  - Fixed recurring fee (monthly or annual).
  - Includes full platform access.
  - Includes hosted LLM usage up to a defined limit.
- Included in tier:
  - Product updates.
  - Standard support.
  - All core features.

### Tier B: Enterprise (Modular)
- Base Platform License:
  - Fixed annual fee for core platform IP.
  - Deployment on customer cloud/on-prem or vendor-managed cloud.
- LLM Flexibility:
  - Option A: Customer integrates own LLM stack (no usage fee from Resonance).
  - Option B: Customer uses Resonance-hosted LLM services (additional usage-based fee).
- Optional Services:
  - Integration consulting.
  - Custom workflow implementation.
  - Ongoing advisory/support retainers.

## 6) Packaging and Commercial Notes
Recommended commercial framing:
- Product fee for platform value.
- Separate usage fee only when hosted inference is provided by Resonance.
- Services sold as optional accelerators, not mandatory for adoption.

This creates:
- Predictable recurring revenue.
- Clean separation between software license and variable compute cost.
- Procurement flexibility for enterprise buyers.

## 7) Next Pricing Decisions Needed
To move from "initial idea" to launch-ready pricing, define:
1. Included usage thresholds for Individual/SMB tier.
2. Overage mechanics and per-unit metering model for hosted LLM usage.
3. Enterprise minimum annual contract value and seat/workspace assumptions.
4. Support SLAs by tier.
5. Service rate card and packaging (implementation package vs day rate vs retainer).

## 8) Proposed Messaging Summary
Resonance is an AI-native workbench for consultants and knowledge teams that unifies memory-aware AI assistance, terminal execution, and documentation in one secure, model-agnostic environment, with pricing that scales from individual operators to modular enterprise deployments.
